# AMESEA：ネオン深海のユウタ

『AMESEA』は、深海に落ちたユウタ・アメシーを操作するネオン深海アクション・ローグライトです。敵を倒してレベルアップし、3択強化と夢ツリーを組み合わせながら、毎回構成の違う深海へ潜ります。

第3開発では4つの深海ゾーン、4種類の新敵、700mボス、5つのビルドシナジー、ランダムミッション、実績、難易度、深海図鑑、難易度別記録を追加しました。外部ライブラリや外部素材は使用していません。

## 起動方法

1. `index.html` をダブルクリックします。
2. Chrome、Edge、Firefoxなどの現行ブラウザで開きます。
3. 難易度を選び、「深海へダイブ」を押します。

Webサーバー、インストール、ビルド作業は不要です。すべてのファイルを同じフォルダー構成のまま置いてください。

## 操作方法

- `WASD` または `矢印キー`：移動
- マウス：照準
- 左クリック または `Space`：ネオン弾
- `Shift`：ネオン・ダッシュ（短時間無敵）
- `E`：ポジティブ・ネオン
- `Q`：自意識バースト（ゲージ100%時）
- `R`：リザルト画面から同じ難易度でもう一度潜る

## 基本ループ

敵を倒すとネオン欠片（経験値）が落ちます。レベルアップするたびに、そのプレイ中だけ有効な強化を3択から1つ選びます。攻撃、連射、弾数、移動、HP、吸引、ネオン、ダッシュなどを組み合わせ、シナジー発動を狙います。

夢の欠片はプレイをまたいで保持され、夢ツリーの永続育成に使えます。家族の記憶はHPを回復し、守護系ビルドでは追加効果を発生させます。

## ポジティブ・ネオン

`E` で発動します。一定時間、光量・攻撃力・視界・アイテム吸引力が上昇し、発動時は近くの敵を怯ませます。

光るほど強くなる一方、敵の索敵やミラーネオンの複製率が上がります。「ネオン過剰」シナジー中は光量に応じてさらに攻撃力が増えますが、敵の出現も増えます。

## 自意識バースト

ゲージ100%時に `Q` で発動します。発動時に周囲へダメージとノックバックを与え、6秒間、火力・連射・移動・防御を強化します。

敵撃破、ネオン中の連続撃破、敵のそばでのダッシュ回避などでゲージが増えます。バースト中の累計撃破は実績にも記録されます。

## 深度とゾーン

| 深度 | ゾーン | 主な特徴 |
|---|---|---|
| 0m〜299m | ネオン浅層 | 明るい初心者向け区間。基本敵とネオン欠片が多い |
| 300m〜699m | 沈黙中層 | 視界が狭く、敵の反応が速い。静寂の層とクラゲ系が増える |
| 700m〜1199m | 夢喰い深層 | 夢喰いと夢の欠片が増え、不規則な敵やイベントが多い |
| 1200m〜 | 家族記憶の最深層 | 高難度。ボス級の圧力、家族の記憶、強い暗闇 |

ゾーン切り替え時は名称と説明を表示します。背景色、視界、敵の出現比率、追加ドロップ、イベント傾向がゾーンごとに変わります。設定は `phase3.js` の `ZONES` に集約されています。

## 敵一覧

- ネガフィッシュ：プレイヤーへまっすぐ迫る基本敵
- 承認欲求クラゲ：光量が高いほど反応が速い
- 夢喰いニュウドウ：間合いを不規則に変え、夢の欠片を落としやすい
- ミラーネオン：少し前のプレイヤー位置を追跡。高光量では複製を作る。撃破でバーストを多く得る
- 水圧アンコウ：遠距離で予兆線を出し、水圧弾を発射。深いほど攻撃が速い
- ネオン喰いウツボ：落ちたネオン欠片と夢の欠片を食べる。撃破すると食べた分を放出
- 深海ピエロクラゲ：ワープと偽物で狙いを惑わせ、確率でレアアイテムを落とす

敵は最大数を制限し、複製は低HP・報酬なしにすることで負荷と無限稼ぎを抑えています。

## ボス一覧

### 承認欲求クラゲ・キング（300m）

放射弾、配下召喚、接触攻撃を使います。光量が高いほど攻撃が激しくなります。

### 夢喰いニュウドウ・アビス（700m）

夢の欠片吸引、暗闇フィールド、螺旋弾、不規則な予兆突進を使います。HP48%以下で暴食形態へ移行し、攻撃間隔と突進が強化されます。撃破すると大量の夢の欠片と家族の記憶を落とします。

## 深海イベント

- 暗流：画面内の存在を横方向へ押し流す
- ネオン嵐：敵出現が増え、アイテムも多く流れ込む
- 静寂の層：視界と敵出現が抑えられる。沈黙中層で起きやすい
- 夢の泡群：ネオン欠片が現れ、夢喰いも混ざる

深層ほどイベント間隔が短くなり、ゾーンに応じて抽選比率が変化します。

## 夢ツリー

夢の欠片を消費する永続育成です。

- ローランドルート：ネオン持続、火力、怯み
- イケメンルート：ダッシュ再使用、無敵、回避
- バカルート：衝撃波、爆発、瀕死火力
- 優しさルート：家族の記憶、最大HP、致命傷防止
- 深海生物ルート：視界、吸引、レアドロップ

夢ツリーの取得状況は、プレイ中のシナジー条件にも加算されます。

## ビルドシナジー

強化カテゴリを2点以上そろえると発動します。夢ツリーの永続強化と、そのプレイ中のレベルアップ強化を合算します。

- ネオン過剰：光量に応じて攻撃力最大+35%。敵出現も増加
- バカ突進：ダッシュ終了時に爆発。敵撃破でダッシュ再使用を短縮
- 優しさ守護：瀕死時に自動バリア。家族の記憶で周囲の敵を減速
- イケメン回避：敵の近くをダッシュするとカウンター弾を発射し、バーストを獲得
- 深海適応：視界悪化を軽減し、レアアイテム率を上昇

発動中のシナジーは右下HUDとリザルトに表示されます。定義は `phase3.js` の `SYNERGIES` に集約されています。

## ミッション

各ダイブ開始時に8種類から重複なしで3つ選ばれます。

- 300m到達
- ネガフィッシュ30体撃破
- ポジティブ・ネオン中に10体撃破
- ダッシュで敵を5体すり抜ける
- 家族の記憶を2個取得
- 夢の欠片を3個取得
- ボスを1体撃破
- HP30%以下で30秒生存

進捗は左下HUDに表示されます。達成時は夢の欠片2〜3個と少量の経験値を獲得します。達成状況はリザルトにも残ります。

## 実績

タイトルの「実績」から確認できます。13個を実装しています。

- 100m / 300m / 1000m到達
- 累計100体 / 1000体撃破
- 夢ツリー初取得
- 承認欲求クラゲ・キング撃破
- 自意識バースト中に累計50体撃破
- 家族の記憶を累計20個取得
- 1プレイで夢の欠片を10個取得
- 夢喰いニュウドウ・アビス撃破
- DEEPで700m到達
- 1プレイで5シナジー発動

解除時は通知され、実績ごとに夢の欠片1〜4個を獲得します。

## 難易度

- CASUAL：敵HP・速度・攻撃・出現を抑制。プレイヤーHPと経験値効率を増加。夢報酬は控えめ
- NORMAL：第2開発に近い標準バランス
- DEEP：敵HP・速度・攻撃・出現を増加。レアドロップと夢報酬を強化

最高深度、最多撃破、最長時間は難易度別に保存されます。リザルトには今回の難易度と、その難易度での最高深度を表示します。

## 深海図鑑

タイトルの「深海図鑑」から開きます。敵、ボス、アイテム、深海イベント、シナジーの5タブがあります。

一度遭遇した項目だけ名前と説明を表示します。未遭遇項目は「？？？」で表示され、遭遇情報はブラウザに保存されます。

## セーブデータ

ブラウザの `localStorage` を使用します。

- `amesea_neon_abyss_save_v2`：第2開発から継続する夢の欠片、夢ツリー、累計記録、ボス記録
- `amesea_neon_abyss_phase3_v1`：実績、遭遇済み図鑑、難易度別記録、長期統計、最後に選んだ難易度

第3開発は追加データを別キーに保存するため、既存セーブをそのまま引き継げます。「セーブデータ初期化」は両方のキーを初期化します。

同じフォルダーを別ブラウザや別PCで開いた場合、セーブは自動共有されません。ブラウザのサイトデータ削除でも消去されます。

## ファイル構成

```text
AMESEA/
├─ index.html   # 画面構造、HUD、タイトル、実績・図鑑、リザルト
├─ style.css    # 第2開発までの基本デザイン
├─ main.js      # 基本アクション、既存敵、アイテム、夢ツリー、300mボス
├─ phase3.css   # 第3開発のHUD、難易度、通知、図鑑、レスポンシブ調整
├─ phase3.js    # ゾーン、新敵、700mボス、シナジー、ミッション、実績、図鑑、難易度
└─ README.md    # このファイル
```

`phase3.js` は既存コードを上書きせず、既存クラスへ機能を追加する構成です。ゾーン、難易度、ミッション、実績、図鑑、シナジーは配列または設定オブジェクトとしてまとめています。

## バランス方針

- CASUALは敵HP・火力・出現を約15〜28%抑え、300mへ到達しやすく調整
- NORMALの300mと700mを装備構成の確認地点として設定
- DEEPは夢ツリー育成を前提に、敵出現と報酬を同時に増加
- ミッション報酬は1回2〜3欠片に制限
- 強いシナジーには出現増加、条件点数、クールダウンなどの制約を設定
- 敵は108体、パーティクルは既存上限420を目安に制御
- ミラーネオンの複製は低HP・報酬なし
