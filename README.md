# AMESEA：ネオン深海のユウタ

外部ライブラリ・外部音源を使わず、HTML / CSS / JavaScriptだけで遊べる深海アクションローグライトです。

深海に落ちたユウタ・アメシーが、自己肯定感から生まれるネオンを武器に、夢と家族の記憶を拾いながら浮上を目指します。明るくなるほど強くなりますが、光を見た敵も寄ってきます。

## 起動方法

1. `index.html` をダブルクリックします。
2. タイトル画面で難易度を選びます。
3. 初めて遊ぶ場合は「チュートリアル」がおすすめです。
4. 「深海へダイブ」で通常プレイを開始します。

ビルド、インストール、ローカルサーバーは不要です。主要デスクトップブラウザの最新版を推奨します。

## 操作方法

| 操作 | キー |
| --- | --- |
| 移動 | `WASD` / 矢印キー |
| 通常攻撃 | マウス左クリック / `Space` |
| ネオン・ダッシュ | `Shift` |
| ポジティブ・ネオン | `E` |
| 自意識バースト | `Q` |
| 一時停止 | `Esc` |
| リザルトから再挑戦 | `R` |

ゲーム中の `Esc` メニューから、操作説明をいつでも確認できます。

## チュートリアル

タイトル画面の「チュートリアル」から始める独立モードです。通常モードの記録や難易度進行とは分離され、敵の攻撃力を抑えています。

全10レッスンを実際に操作して進めます。

1. 移動する
2. 攻撃する
3. ダッシュする
4. ネオン欠片を拾う
5. ポジティブ・ネオンを使う
6. 光るほど敵が寄ることを観察する
7. 自意識バーストを使う
8. 家族の記憶を拾う
9. レベルアップ3択強化を選ぶ
10. 小型ボス「ネガフィッシュ・キャプテン」を倒す

初回クリア時は夢の欠片3個、再挑戦時は1個を獲得します。クリア状況は保存され、タイトルに「チュートリアル済み」と表示されます。

## ポジティブ・ネオン

`E` で一定時間発動します。

- 攻撃力とアイテム吸引力が上昇
- 周囲の敵を押し返す
- 光量が増え、探索しやすくなる
- 敵の索敵範囲と出現圧も上昇

「強くなるほど見つかる」がコアのリスク・リターンです。

## 自意識バースト

敵撃破、危険なダッシュ回避などでBURSTゲージが増えます。100%時に `Q` で発動します。

- 一時的に攻撃、移動、防御を強化
- 発動時に周囲へダメージとノックバック
- 発動中は弾が強化され、貫通が付く
- 終了後は短い残光状態になる

## 深度とゾーン

時間経過で深度が増え、敵、視界、イベント、背景色が変化します。

| 深度 | ゾーン |
| --- | --- |
| 0〜299m | ネオン浅層 |
| 300〜699m | 沈黙中層 |
| 700〜1199m | 夢喰い深層 |
| 1200m〜 | 家族記憶の最深層 |

300mと700mにはボスが出現します。1200m以降は家族の記憶が見つかりやすい一方、強敵比率も上がります。

## 敵一覧

| 敵 | 特徴 |
| --- | --- |
| ネガフィッシュ | 基本敵。群れで接近 |
| 承認欲求クラゲ | 光に敏感。明るいほど加速 |
| 夢喰いニュウドウ | 間合いを取り、夢の欠片を落としやすい |
| ミラーネオン | 少し前のプレイヤー位置を追跡 |
| 水圧アンコウ | 予兆線のあとに高速弾を発射 |
| ネオン喰いウツボ | 落ちている欠片を奪う |
| 深海ピエロクラゲ | ワープと偽物で狙いを惑わせる |

遭遇した敵は「深海図鑑」に記録されます。

## ボス一覧

- **承認欲求クラゲ・キング**：300mで出現。弾幕と配下召喚を使います。
- **夢喰いニュウドウ・アビス**：700mで出現。吸引、暗闇、突進、螺旋弾を使います。
- **ネガフィッシュ・キャプテン**：チュートリアル最後の小型ボスです。

ボスと通常敵が同時に存在する場合も、弾の命中・死亡処理・報酬処理を個別に行います。

## 深海イベント

- **暗流**：海流が全員を横へ押します。
- **ネオン嵐**：敵と欠片が増える危険な稼ぎ時です。
- **静寂の層**：視界と敵出現が減ります。
- **夢の泡群**：経験値と夢喰いが連続出現します。

一時停止、レベルアップ、設定画面中はイベント時間も停止します。

## 夢ツリー

夢の欠片を使う永続育成です。5ルート、各3段階です。

- ローランド：ポジティブ・ネオン
- イケメン：回避とカウンター
- バカ：突進と爆発
- 優しさ：回復、守護、家族
- 深海生物：視界、水圧耐性、探索

上位ノードは同ルートの手前のノードを取得すると購入できます。

## ビルドシナジー

同系統の夢ツリーやレベルアップ強化を2つ以上集めると発動します。

- ネオン過剰
- バカ突進
- 優しさ守護
- イケメン回避
- 深海適応

発動履歴はリザルトに表示され、長期クエストにも加算されます。

## ミッション

各ダイブでランダムに3つ選ばれる短期目標です。

深度到達、特定敵撃破、ネオン中の撃破、ダッシュ回避、記憶・夢の回収、ボス撃破、低HP生存などがあります。達成時に夢の欠片と経験値を即時獲得します。

## 実績

累計撃破、深度、ボス、バースト、家族の記憶、シナジーなど13種類です。条件達成時に自動解放され、夢の欠片を獲得します。報酬は一度だけです。

## 長期クエスト

タイトルの「クエスト」から確認します。複数回のダイブにまたがり進行し、達成後に「報酬受取」を押す方式です。

| クエスト | 条件 | 報酬 |
| --- | --- | --- |
| ネガティブの大群 | ネガフィッシュ200体 | 夢の欠片6 |
| 夢の密漁ではない | 夢の欠片50個回収 | 夢の欠片8 |
| 家族の電波、良好 | 家族の記憶30個 | 永続強化：最大HP+10 |
| 承認は足りています | クラゲ・キング3回 | 夢の欠片7 |
| 夢喰い、おかわり | アビス2回 | 夢の欠片10 |
| 高水圧デビュー | DEEPで300m | 記憶ログ解放 |
| 全部、つながっている | シナジー累計20回 | 永続強化：バースト獲得+8% |
| 準備はだいたい完璧 | チュートリアルクリア | 夢の欠片3 |

進行中、達成済み、受取済みを別表示します。受取状態を先に保存してから報酬を適用し、連打による二重受取を防ぎます。

## 深海記憶ログ

タイトルの「記憶ログ」から読める短い物語です。未解放は「？？？」と解放条件を表示します。

1. 深海に落ちた日
2. 光れば見える。光れば狙われる。
3. 承認欲求クラゲ・キング
4. 夢を食うもの
5. 家族の声
6. 自意識は、鎧にもなる
7. バカだから進める場所
8. 帰る場所
9. ネオンは消えない
10. 深海？ いや、ここは俺のステージ。

深度、ボス撃破、家族の記憶、夢ツリー、DEEP記録で解放されます。新規解放は通知とリザルトに表示します。

## 深海図鑑

敵、ボス、アイテム、深海イベント、シナジーを分類表示します。未遭遇項目は名前と説明を伏せ、実際に遭遇すると保存されます。

## 難易度

- **CASUAL**：敵HP・速度・攻撃・出現を抑制。プレイヤーHPと経験値を増加
- **NORMAL**：標準バランス
- **DEEP**：敵と出現圧を強化。夢・レア報酬率を上昇

難易度別の最高深度、撃破数、プレイ時間を個別保存します。

## 設定

タイトルまたは一時停止メニューから開けます。変更は即時反映・自動保存されます。

- マスター音量
- BGM音量
- 効果音音量
- ミュート
- 画面揺れの強さ
- パーティクル量（少ない / 標準 / 多い）
- ダメージ表示 ON / OFF
- 色覚サポートモード
- 自動攻撃 ON / OFF
- 設定を初期値へ戻す
- 全セーブデータ初期化（確認あり）

## サウンド

外部音源ファイルは使っていません。初回のクリックまたはキー入力後にWeb Audio APIを開始し、オシレーターで音を生成します。

効果音：通常攻撃、敵撃破、アイテム取得、レベルアップ、ポジティブ・ネオン、自意識バースト、ダッシュ、被ダメージ、ボス出現、実績解除、クエスト達成、UI決定。

BGMは深海の低音とネオンの電子音を自動生成します。ボス戦は低音を強め、バースト中は高音の密度を上げます。AudioContextが使えない、または再生がブロックされた環境でもゲーム処理は継続します。

## アクセシビリティ

- **自動攻撃**：射程内で最も近い敵へ通常攻撃。手動攻撃も併用可能
- **色覚サポート**：敵・アイテムへ形状記号、味方弾へ山形、敵弾へ×印を追加
- **画面揺れ調整**：0〜100%。0で実質OFF
- **パーティクル量**：視認性と負荷に合わせて変更
- **ダメージ表示**：必要に応じて非表示
- **重要通知**：最大3件に制限し、実績・クエスト・記憶ログを分類
- **操作説明**：ゲーム中の一時停止メニューから常時確認
- **小画面対応**：タイトル、各一覧、設定をスクロール可能にし、カードを1列へ再配置

## セーブデータ

localStorageへ自動保存します。

- `amesea_neon_abyss_save_v2`：夢の欠片、夢ツリー、基礎累計記録
- `amesea_neon_abyss_phase3_v1`：難易度別記録、実績、図鑑、追加統計
- `amesea_neon_abyss_phase4_v1`：チュートリアル、記憶ログ、長期クエスト、設定、永続クエスト報酬

各層を個別に検証して読み込みます。JSON破損、型不正、保存不可時は安全な初期値へフォールバックし、ゲームを継続します。

設定画面または夢ツリーの「セーブデータ初期化」は3層すべてを初期化します。確認ダイアログが表示されます。

## 一時停止

`Esc` で開きます。

- 再開
- 操作説明
- 設定
- タイトルへ戻る（確認あり）

一時停止中はゲーム更新が走らないため、ゲーム時間、敵、弾、深海イベント、チュートリアル目標が停止します。設定から戻っても同じ状態で再開します。

## パフォーマンスと安定性

- 敵、味方弾、敵弾、アイテム、パーティクルに上限
- パーティクル設定に応じた生成数・保持数の調整
- 画面外の敵弾と敵を削除
- 重要通知は最大3件、ダメージ数字は最大28件
- 長期進行のDOM更新を間引き
- `requestAnimationFrame` をゲームループに使用
- レベルアップ、一時停止、設定中は更新停止
- 開始時・タイトル復帰時に前回オブジェクトを破棄
- ボスと通常敵の命中対象を分離
- 報酬受取済みフラグで二重受取を防止
- 音声・localStorageを例外処理で保護

## ファイル構成

| ファイル | 内容 |
| --- | --- |
| `index.html` | Canvas、HUD、タイトル、夢ツリー、実績、図鑑、リザルト |
| `style.css` | 基本レイアウトとゲームUI |
| `main.js` | 基本アクション、敵、ボス、アイテム、永続育成、セーブ |
| `phase3.css` | 第3開発のゾーン、難易度、図鑑、ミッションUI |
| `phase3.js` | 第3開発の敵、700mボス、シナジー、実績、図鑑 |
| `phase4.css` | 第4開発のメニュー、設定、クエスト、チュートリアル、アクセシビリティ |
| `phase4.js` | 第4開発の全ロジック、Web Audio、安定化・最適化 |
| `README.md` | 本仕様書 |

読み込み順は `main.js` → `phase3.js` → `phase4.js` です。第4開発は既存機能を上書き削除せず、既存クラスをラップして機能を追加します。
