# AMESEA：ネオン深海のユウタ

『AMESEA』は、深海に落ちたユウタ・アメシーを操るネオン深海アクション・ローグライトです。敵を倒し、夢と家族の記憶を集めながら潜り続けます。

中心にあるのは「光るほど強くなるが、光るほど危険になる」というリスク＆リターンです。1プレイ内のランダム強化に加え、夢の欠片を使う永続育成、300mのボス戦、ランダムな深海イベントを実装しています。

## 起動方法

1. `index.html` をダブルクリックします。
2. Chrome、Edge、Firefoxなどの現行ブラウザで開きます。
3. 「深海へダイブ」をクリックすると開始します。

インストール、Webサーバー、外部ライブラリ、画像素材は不要です。

## 操作方法

- `WASD` または `矢印キー`：移動
- マウス：照準
- 左クリック または `Space`：ネオン弾
- `Shift`：ネオン・ダッシュ（短時間無敵）
- `E`：ポジティブ・ネオン
- `Q`：自意識バースト（ゲージ100%時）
- `R`：ゲームオーバー時にもう一度潜る

## コアシステム

敵を倒すと経験値アイテムが出現します。経験値が規定値に達するとレベルアップし、そのプレイ中だけ有効な3択強化を1つ選びます。攻撃力、連射、弾数、移動、HP、吸引、ネオン、バーストなどを組み合わせてビルドを作れます。

敵の撃破、ネオン中の連続撃破、敵のそばでのダッシュ回避によりバーストゲージが増加します。夢の欠片は永続通貨、家族の記憶は回復・守護アイテムです。

## ポジティブ・ネオン

`E` で一定時間、光量・攻撃力・視界・アイテム吸引力が上昇します。発動時は周囲の敵を怯ませます。一方、光量が高いほど敵の索敵範囲や一部の敵の速度が上がり、危険も増します。

夢ツリーのローランドルートで持続時間、ネオン中の攻撃力、発動時の怯み時間を永続強化できます。

## 自意識バースト

ゲージ100%時に `Q` で発動します。発動時に周囲へダメージとノックバックを与え、6秒間、火力・連射・移動・防御を大幅強化します。画面フラッシュ、リング、ネオン色の変化で発動を表示します。

## 深度

深度は時間とともに増え、300mにはおよそ1分で到達します。深度が増すほど背景と視界が暗くなり、敵のHP・速度・強敵率が少しずつ上昇します。永続強化「暗闇と友達」は深度による視界悪化を軽減します。

## ボス

深度300mで「承認欲求クラゲ・キング」が出現します。

- 主人公の光量が高いほど移動速度と攻撃頻度が上昇
- 小型の承認欲求クラゲを召喚
- 円形のネオン弾を発射
- 出現中は通常敵の発生を抑制
- 専用HPバー、警告、背景色、パーティクル演出

撃破すると大量の経験値、夢の欠片、家族の記憶を獲得します。初回撃破はセーブデータの解放記録に保存されます。

## 深海イベント

イベントは1つずつランダムに発生し、開始時に中央告知、開催中は右上に残り時間を表示します。

- 暗流：プレイヤーと敵が横方向へ流される
- ネオン嵐：粒子とアイテムが増えるが、敵も増える
- 静寂の層：敵の出現が減るが、視界も狭くなる
- 夢の泡群：経験値アイテムが大量に現れ、夢喰いニュウドウも増える

ボス出現中は新しいイベントが始まらないため、イベント同士やボス演出が重なりすぎません。

## 夢ツリー

タイトルまたはリザルト画面から開けます。夢の欠片を消費し、各ルートを前から順に取得します。取得効果は次のダイブから反映されます。

- ローランドルート：ネオン持続、ネオン火力、怯み
- イケメンルート：ダッシュ再使用、無敵時間、接触回避
- バカルート：ダッシュ衝撃波、弾の爆発、瀕死火力
- 優しさルート：家族の記憶の回復、最大HP、致命傷耐え
- 深海生物ルート：暗闇適応、吸引範囲、レア率

各ルートは3段階です。未取得の前提、欠片不足、取得済みなど、購入できない理由を各ノードに表示します。

## 家族の記憶

取得時にHPを回復し、家族にまつわる短いメッセージをランダム表示します。優しさルートにより回復量が増え、最終段階では一時防御と致命傷耐えの回復も得られます。

## セーブデータ

ブラウザの `localStorage` にキー `amesea_neon_abyss_save_v2` で自動保存します。

保存内容：

- 所持中の夢の欠片
- 夢ツリーの取得状況
- 累計最大深度
- 累計撃破数
- 累計プレイ回数
- ボス初回撃破などの解放状況
- 音量設定用の値
- 最高撃破数、最長プレイ時間、ボス撃破数

夢の欠片取得時、一定数の敵撃破時、記録更新時、永続強化購入時、ゲームオーバー時、ページ非表示時に保存します。読み込み時は値と強化IDを検証し、JSON破損・不正な形式・異なるバージョンの場合は安全な初期データへ戻します。

夢ツリー下部の「セーブデータ初期化」から全データを消去できます。誤操作防止の確認ダイアログがあります。`file://` ページの保存領域はブラウザごとに管理されるため、別ブラウザへは引き継がれません。

## リザルト

今回と最高の深度、今回と累計の撃破数、今回と所持中の夢の欠片、レベル、プレイ時間、評価ランクを表示します。「もう一度潜る」「夢ツリーへ」「タイトルへ」を選べます。

## ファイル構成

- `index.html`：Canvas、HUD、ボスHP、タイトル、3択強化、夢ツリー、リザルト
- `style.css`：深海ネオンのUI、警告、画面演出、レスポンシブ表示
- `main.js`：保存、入力、描画、主人公、敵、ボス、弾、アイテム、イベント、強化、ゲームループ
- `README.md`：ゲーム仕様とセーブ仕様

`main.js` は依存関係を増やさず `index.html` の直接起動を保つため1ファイルですが、機能ごとのクラスとセクションコメントで分離しています。敵データ、夢ツリーデータ、ボスクラス、イベント定義を独立して追加しやすい構造です。
