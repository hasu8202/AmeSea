# AMESEA：ネオン深海のユウタ

深海ネオン・リアルタイムアクションRPGのMVPです。主人公「ユウタ・アメシー」を操り、夢と家族の記憶を集めながら、暗い深海を生き延びます。

中心にあるのは「光るほど強くなるが、光るほど危険になる」というリスク＆リターンです。ポジティブ・ネオンで視界、攻撃力、アイテム吸引を強化できますが、敵の発生数・索敵距離・強敵率も上昇します。

## 起動方法

1. `index.html` をダブルクリックします。
2. Chrome、Edge、Firefoxなどの現行ブラウザで開きます。
3. 「深海へダイブ」をクリックすると開始します。

インストール、Webサーバー、外部ライブラリは不要です。

## 操作方法

- `WASD` または `矢印キー`：移動
- `マウス`：照準
- `左クリック` または `Space`：ネオン弾
- `Shift`：ネオン・ダッシュ（短時間無敵）
- `E`：ポジティブ・ネオン
- `Q`：自意識バースト（ゲージ100%時）
- `R`：ゲームオーバー時にリスタート

## ゲームシステム

- **ネオン光量**：通常時は安全寄り。Eキーで強く光ると視界・火力・吸引力が増しますが、敵も激しく集まります。
- **自意識バースト**：撃破、寸前回避、ネオン中の連続撃破でゲージが増加。発動中は火力・速度・防御が大幅強化され、敵を引き寄せます。
- **敵**：基本型のネガフィッシュ、光に敏感な承認欲求クラゲ、間合いを変える夢喰いニュウドウが出現します。
- **成長**：ネオン欠片で経験値を獲得し、レベルアップ時にランダムな3つの強化から1つを選びます。
- **深度**：時間とともに深度が増え、背景が暗く、敵が強く多くなり、レアアイテム率も上昇します。
- **アイテム**：経験値のネオン欠片、レア素材の夢の欠片、回復できる家族の記憶があります。
